# Design reference — Public, Autopilot, Coinbase → Thesis

Reference screenshots live in `assets/IMG_6066–6070.png`. Claude Code: **adapt patterns, do not clone apps**. Thesis stays **Daylight** (light surfaces, teal brand), **educational**, **not** brokerage or copy-trading.

---

## What to steal (patterns)

### 1. Public / Coinbase — **Insight narrative** (priority)

| Pattern | Reference | Thesis adaptation |
|---------|-----------|-------------------|
| One-sentence headline | “Markets are rising today…” | Stock/theme **context headline** (mock, illustrative) |
| Attribution line | “Generated by Agents at 2:12 PM” | **“Thesis lens · illustrative”** or “Based on your profile frame” — never imply live AI advice |
| Body with **linked terms** | Blue `SPY`, `P/E` links | `TermLink` → `ExplainSheet` (brand teal, not iOS blue) |
| **Why it matters** | Bulleted causes | Map to education — cause/effect teaching |
| **Watch** | Counter-risk paragraph | Map to **Stock signals** / due diligence (E8–E10) |
| Ticker **chips** | `SPY +0.22%` pills | Mock chips on insight footer (optional %, demo only) |
| **Bottom sheet** deep dive | Coinbase “Happening Now” expand | `InsightSheet` — full narrative on tap “Read more” |

**Thesis framing:** Explain price action and context **for learning**, not “what to trade today.”

### 2. Autopilot — **Minimal list rows**

| Pattern | Reference | Thesis adaptation |
|---------|-----------|-------------------|
| Circular lead visual | Creator photo | Theme **glyph** in tinted circle (`theme.color` bg) |
| Performance pill | `+40.0%` · 1 Year | **Not** returns-led (avoid implying performance promises). Use **heat** or **match %** pill: `Top match`, `Long arc`, or `4 reasons` |
| Title + subtitle | Name + “Autopilot” | Theme **title** + **kicker** or `author` |
| Chevron | `>` | Keep — navigate to theme/stock detail |
| Section + “Explore more” | Footer link | “All themes” / functional search later |

Use on **Themes tab** (`themes.tsx`) and optionally **concept library** list (future).

### 3. Public — **Markets feed density**

| Pattern | Reference | Thesis adaptation |
|---------|-----------|-------------------|
| Date + session kicker | “Friday, May 29 · Pre Market” | Optional on Home: “Your frame · updated when you duel” |
| Rows in subtle rounded rects | Ticker list | Watchlist rows or insight-related tickers |
| Bottom nav “Agents” | Dedicated AI tab | **No separate Agents tab** — insights live on Home, Stock, Duel |

---

## What NOT to copy

- Celebrity / politician **copy portfolios** (Pelosi Tracker, etc.)
- **Dark mode** as default (Thesis = Daylight; dark insight card variant optional later)
- **Live** market AI or real-time “Agents” without data license
- Green/red performance as **primary** hook (educational app, not leaderboard)
- Language that sounds like **recommendations** or **autopilot investing**

## Thesis design tokens (use these, not screenshot colors)

```text
Background:     bg (#F3F5F1), bg-surface (#FFFFFF)
Text:           ink, ink-2, ink-3
Brand / links:  brand (#0E7A66) — hyperlinks & TermLink
Positive chip:  pos / pos-bg (mock +% only with disclaimer)
Notice:         amber / amber-bg (signals)
Cards:          border-line, rounded-[14px]–[16px], generous py
Type:           font-displayX (titles), font-sansMd (body), font-sansX (kickers)
```

**Link style:** `text-brand font-sansSb` + underline on press — not `#007AFF` unless adding `link` token to `tailwind.config.js`.

### `InsightCard` props (spec)

```ts
type InsightCardProps = {
  kicker?: string;           // "Friday · Illustrative"
  headline: string;
  attribution?: string;      // "Thesis lens · demo"
  body?: string;             // short paragraph; TermLinks inside
  whyItMatters?: string[];   // bullets
  watch?: string;            // single caution / signal paragraph
  chips?: { label: string; delta?: string; tone?: "pos" | "neg" | "neutral" }[];
  onPressReadMore?: () => void;
};
```

### `ListRow` props (spec)

```ts
type ListRowProps = {
  leading: ReactNode;        // glyph circle
  badge?: { text: string; tone?: "brand" | "pos" | "amber" };
  title: string;
  subtitle?: string;
  onPress: () => void;
};
```

## Mock content rules

- All insights/signals: suffix **“Illustrative · not live data”**
- Headlines describe **mechanics** (“Volatility is elevated vs its 1Y average”) not **commands** (“Buy the dip”)
- CEO churn / alerts → **StockSignal** (E8), styled as **Watch** block inside `InsightCard`, not red alarm UI
